# FitForge 
AI-powered Personal Fitness Coach built with 
Groq API + Streamlit + Python

## Features
- Personalized workout & diet plans
- Daily progress tracking
- AI plan modification based on feedback
- Multi-user support
- Progress charts & history

## Tech Stack
- Groq API (Llama 3.3-70b)
- Streamlit
- Python
- Pandas

## Setup
1. Clone the repo
2. Install requirements
   pip install -r requirements.txt
3. Create .env file
   GROQ_API_KEY=your_key_here
4. Run
   streamlit run app.py
